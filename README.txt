El programa se compone de los siguientes elementos:

    - main.ipynb: jupyter notebook que presenta el ejercicio de clasificación de imágenes realizado.
    - imshow.py: script de python que define la función imshow() para graficar imágenes que han sido 
                 inicialmente transformadas para procesar en el modelo de redes neuronales.
    - Net.py: script de python que define la clase Net() que contiene un modelo de redes neuronales 
              convusionales que recibe originalmente imágenes de 64 px x 64 px.
